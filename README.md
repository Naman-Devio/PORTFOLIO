# Naman Vaishnav Portfolio v2.0
Hello there! I'm Naman Vaishnav, a B.Tech Computer Science student, builder of bots, APIs, and interactive web experiences.

This is the interactive 3D version of my personal portfolio, built to showcase my journey, hackathon projects, and technical skills.

Checkout the live version at [check](https://naman-pf.vercel.app/)

## 🚀 Tech Stack

- **Framework**: Next.js
- **Library**: React
- **3D Engine**: React-three-fiber & Three.js
- **Utilities**: DREI
- **Animations**: GSAP
- **State Management**: Zustand
- **Styling**: Tailwind CSS
